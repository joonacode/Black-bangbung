Tentang Game

The Bangbung Game adalah game sederhana yang dibuat dengan menggunakan software Construct 2. Game ini terinspirasi dari hewan kecil, yaitu bangbung. Game ini akan sangat membutuhkan kesabaran dari para pemain karena tingkat kesusahan yang dimiliki oleh game tersebut. Gameplay nya sendiri tergolong sangat mudah dipahami karena kalian hanya perlu menekan gambar panah yang ada pada bagian kanan bawah game. Dengan tombol tersebut kalian akan memerintahkan Bangbung yaitu hewan yang digunakan sebagai actor pada game ini untuk bergerak agar dapat menghindari berbagai rintangan yang akan dihadapi oleh bangbung itu sendiri.
